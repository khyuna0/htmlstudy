1. 웹 · 네트워크 핵심 요약
웹이란?
인터넷에서 정보를 주고받는 서비스 플랫폼

하나의 사이트가 아니라, 전체 구조를 의미함

오프라인 정보를 온라인으로 옮기는 작업 = 디지털 전환

네트워크란?
여러 컴퓨터가 서로 연결된 구조

네트워크와 네트워크를 연결하는 장치 = 라우터

전 세계 수많은 라우터와 네트워크가 연결된 것 = 인터넷

서버와 클라이언트
서버: 정보를 저장하고 요청을 처리하는 컴퓨터

클라이언트: 서버에 요청하는 컴퓨터 (브라우저)

웹 = 클라이언트 ↔ 서버 간 통신 구조

주소 관련 개념
IP 주소: 컴퓨터의 실제 주소 (숫자)

도메인: IP를 사람이 보기 쉽게 만든 이름 (예: naver.com)

DNS 서버: 도메인 → IP 주소로 바꿔주는 시스템

HTTP / HTTPS
HTTP: 웹 통신을 위한 규칙(프로토콜)

HTTPS: HTTP에 **보안(암호화)**이 추가된 버전

2. 2진수 · 진수 변환 요약
진수 개념
2진수 (Binary): 0과 1만 사용

10진수 (Decimal): 우리가 평소 사용하는 숫자

16진수 (Hex): 09 + AF (예: FF, 2A)

변환 방법
2진수 → 10진수
각 자리의 값 × 2ⁿ (n = 자리수, 오른쪽부터 0)

예: 1011 → 1×2³ + 0×2² + 1×2¹ + 1×2⁰ = 8 + 0 + 2 + 1 = 11

10진수 → 2진수
2로 나누고 나머지를 거꾸로 읽음

예: 13 → 1101

2진수 → 16진수
오른쪽부터 4자리씩 묶어서 변환

예: 10101100 → 1010(=A), 1100(=C) → AC

3. HTML 태그 요약 정리
웹 문서 기본 구조
<!DOCTYPE html>: HTML5 문서 선언

<html>: 문서 전체 영역

<head>: 설정 정보 (제목, 문자 인코딩 등)

<body>: 화면에 실제로 보이는 영역

텍스트 관련 태그
<h1> ~ <h6>: 제목 (숫자가 작을수록 크고 굵음)

<p>: 단락 (줄 바꿈 포함)

<br>: 줄 바꿈 (닫는 태그 없음)

<hr>: 가로줄

이미지 관련 태그
<img src="경로" alt="설명">: 이미지 삽입

width, height: 크기 조절

<figure> + <figcaption>: 이미지 + 설명 묶기

링크 관련 태그
<a href="주소">: 하이퍼링크

외부 사이트: https:// 붙이기

내부 페이지: 파일명.html

target="_blank": 새 탭에서 열기

#id값: 같은 페이지 안에서 이동

멀티미디어 태그
<audio src="..." controls>: 오디오 삽입

<video src="..." controls autoplay muted poster="썸네일">: 비디오 삽입

<embed> / <object>: PDF, 미디어 파일 등 외부 리소스 삽입

입력 양식 (form)
<form>: 사용자 입력을 서버로 전송
type="text": 일반 텍스트

type="password": 비밀번호

type="email": 이메일 형식

type="tel": 전화번호

type="radio": 하나만 선택 (같은 name끼리)

type="checkbox": 여러 개 선택 가능

type="number": 숫자 입력

type="range": 슬라이더

type="date" / datetime-local: 날짜, 시간

type="file": 파일 업로드

type="submit": 전송 버튼

type="reset": 입력값 초기화

type="button": 자바스크립트 기능 연결

type="image": 이미지 버튼


<input type="radio">

하나만 선택할 때 사용 (ex. 성별, 결제 방식)

name을 같게 하면 묶임

기본 선택은 checked 사용

<input type="checkbox">

여러 개 선택할 때 사용 (ex. 취미, 관심사)

name은 같아도 되고 달라도 됨

기본 선택은 checked 사용
